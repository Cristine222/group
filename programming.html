Introduction to Programming

Simple Programs
Writing instructions that a computer can understand and follow to perform specific tasks.

Programming helps create websites, mobile apps, games, and software systems.

Lectures

1. Variables and Data Types

A variable is like a container or box that holds data which can change during the program. For example, a variable can store a name, a number, or a result of a calculation. It is given a name so the programmer can easily use it later in the code. Data types define what kind of data a variable can store. Common data types include integer (whole numbers like 1, 10, 100), float (numbers with decimals like 3.5 or 7.2), string (text like “Hello”), and boolean (true or false values).

2. Control Structures (if statements, loops)

Control Structures (If Statements and Loops) are important concepts in programming that control the flow of a program or decide how instructions are executed. An if statement is used for decision-making in a program. It checks a condition and runs a specific block of code only if the condition is true. For example, if a number is greater than 10, the program can display a message. If the condition is false, it can skip or run another block of code using else or else if. Loops are used to repeat a block of code multiple times until a condition is met. Common types of loops include the for loop, which runs a set number of times, and the while loop, which continues as long as a condition is true. Loops help save time and reduce repeated code.

3. Functions and Procedures

Functions and Procedures are important concepts in programming that help organize code into reusable blocks of instructions. A function is a block of code that performs a specific task and usually returns a value. For example, a function can calculate the sum of two numbers and give back the result. Functions help make programs shorter, easier to read, and easier to debug because the same code can be used many times without rewriting it. A procedure is similar to a function, but it does not return a value. Instead, it performs an action, such as displaying a message, printing output, or updating data. Procedures are used when you only need to execute steps without needing a result.

4. Arrays and Data Structures

An array is a collection of multiple values stored in a single variable. These values are usually of the same type, such as numbers or strings. Each item in an array has an index number that helps you access it easily. For example, an array can store a list of student names or test scores. Arrays make it easier to handle large amounts of related data without creating many separate variables. Data structures are ways of organizing and storing data so it can be used efficiently in a program. Besides arrays, other common data structures include lists, stacks, queues, linked lists, and dictionaries. Each type is used for different purposes, such as storing ordered data, managing tasks, or quickly searching information.

5.Algorithms and Problem Solving

An algorithm is a clear set of instructions that tells the computer how to complete a task. It must be simple, logical, and follow a specific order. For example, an algorithm for adding two numbers would include steps like: get the first number, get the second number, add them together, and show the result. Problem solving in programming means breaking a big problem into smaller, easier steps so it can be solved efficiently. It involves understanding the problem, planning a solution, writing the algorithm, and then converting it into code..
